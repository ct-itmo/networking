# Компьютерные сети

См. также: [инструкция для Quirck](https://github.com/ct-itmo/quirck/tree/master/README.md)

## Запуск

* Подготовьте Quirck
* `python -m quirck`
* `python -m networking.core.socket`

## Конфигурация

Добавьте в `.env`:

```
SECRET_SEED=...
SOCKET_PATH=/path/to/networking.sock
```
